# Калькулятор зарплаты

Веб-калькулятор для расчёта зарплаты с учётом НДФЛ и страховых взносов работодателя.

## Возможности

- Расчёт суммы на руки из оклада (gross → net)
- Обратный расчёт оклада из суммы на руки (net → gross)
- Ставка НДФЛ: 13% (резиденты) / 30% (нерезиденты)
- Расчёт взносов работодателя: ОПС (22%), ОМС (5.1%), ОСС (2.9%)
- Общие затраты работодателя на сотрудника
- Адаптивный дизайн для мобильных устройств

## Запуск

Откройте `index.html` в браузере.

## Технологии

HTML, CSS, JavaScript — без зависимостей.
