# Калькулятор зарплаты КР

Веб-калькулятор для расчёта зарплаты по законодательству Кыргызской Республики (2026).

## Возможности

- Расчёт суммы на руки из оклада (gross -> net)
- Обратный расчёт оклада из суммы на руки (net -> gross)
- Подоходный налог: 10% (от оклада за вычетом взноса в Соцфонд)
- Взнос работника в Социальный фонд: 10%
- Взносы работодателя в Социальный фонд: 17.25% (пенсионный 15% + ФОМС 2% + ФОТ 0.25%)
- Общие затраты работодателя на сотрудника
- Адаптивный дизайн для мобильных устройств

## Формула расчёта

- Взнос в Соцфонд работника = оклад x 10%
- Подоходный налог = (оклад - взнос в Соцфонд) x 10%
- На руки = оклад - взнос в Соцфонд - подоходный налог

## Запуск

Откройте `index.html` в браузере.

## Технологии

HTML, CSS, JavaScript — без зависимостей.
